# GameOptimizer

**GameOptimizer** is a simple and practical Windows program designed to optimize your system and free up RAM, especially for gamers.

---

## Features
- Close selected running applications  
- Free up RAM used by processes  
- Show operation progress with a Progress Bar  
- Select specific programs to optimize  

---

## How to Use
1. Run the `GameOptimizer.exe` file  
2. Select the programs you want to optimize  
3. Click the **Optimize Selected** button to free RAM and close the selected apps  
4. The progress and status of the operation are displayed in the Progress Bar and the status text at the bottom
